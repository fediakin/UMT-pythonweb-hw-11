# Contacts REST API (v1.1) 📇

REST API для зберігання та управління контактами. Оновлена версія з додаванням системи авторизації, підтвердження email та завантаженням файлів.

## 🚀 Нові функції (ДЗ 11)

- **Аутентифікація та Авторизація:** Логіка реєстрації та логіну з використанням JWT-токенів.
- **Хешування паролів:** Безпечне зберігання паролів у БД за допомогою `bcrypt`.
- **Ізоляція даних:** Користувачі мають доступ лише до власних контактів (зв'язок One-to-Many).
- **Верифікація Email:** Відправка листа з унікальним токеном через `fastapi-mail` для підтвердження пошти.
- **Обмеження запитів (Rate Limiting):** Захист роутера `/users/me` від спаму за допомогою `Redis` та `fastapi-limiter` (максимум 5 запитів на хвилину).
- **CORS:** Налаштовано `CORSMiddleware` для безпечних крос-доменних запитів.
- **Аватарки:** Інтеграція з `Cloudinary` для завантаження та оновлення фото профілю.
